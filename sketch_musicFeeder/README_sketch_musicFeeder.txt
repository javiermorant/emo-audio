This is a Processing File.

Requieres libraries OSCP5 and ControlP5, you must add them tu run the program
The files will be processed assigning a value depending on the file name:
FEAR: 1
SADNESS: 2
HAPINESS: 3
files not containing those tokens will be classified as 4

To run the program put in the text field the path where the audio files are located and press process
